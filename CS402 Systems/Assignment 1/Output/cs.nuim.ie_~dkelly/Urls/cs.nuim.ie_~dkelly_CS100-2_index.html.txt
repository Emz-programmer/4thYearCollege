<p class=MsoNormal><b>Notes on the programming assignments<o:p></o:p></b></p>

<p class=MsoNormal>The <b>first programming assignment</b> tests your understanding
of simple Java data types and operators, expression evaluation and basic
console input/output.</p>

<p class=MsoNormal><![if !supportEmptyParas]>&nbsp;<![endif]><o:p></o:p></p>

<p class=MsoNormal>The <b>second programming assignment</b> tests your
understanding of flow control statements for selection and repetition.</p>

<p class=MsoNormal><![if !supportEmptyParas]>&nbsp;<![endif]><o:p></o:p></p>

<p class=MsoNormal>The <b>third programming assignment</b> tests your
understanding of java classes, objects, methods and parameters by giving you an
existing program which must be extended.</p>

<p class=MsoNormal><![if !supportEmptyParas]>&nbsp;<![endif]><o:p></o:p></p>

<p class=MsoNormal>The <b>fourth programming assignment</b> tests your
understanding of object oriented design methodology and your ability to use
java language structures to implement that design from scratch.</p>

<p class=MsoNormal><![if !supportEmptyParas]>&nbsp;<![endif]><o:p></o:p></p>

<p class=MsoNormal>The <b>fifth programming assignment</b> is a capstone
project which allows you to demonstrate many of the java language features that
you have learnt as well as requiring the implementation and management of more
complex data structures.<b><o:p></o:p></b></p>
