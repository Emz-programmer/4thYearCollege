<CENTER><B>Course Overview</B></CENTER>


<P>An operating system is very large and complex software system which
manages all of the hardware of a computer and makes it easy to use. It
is an environment where numerous tasks must be handled at the same time
and the coordination of these tasks must provide for the most efficient
and best use of available resources.

<P>Among some of the tasks an operating must perform are:-
<UL>(i) Allocation and Management of Resources among competing user processes.
<BR>(ii) Maximising Resource Utilisation with the goal of improving overall
system throughput.
<BR>(iii) Providing a user interface between the user and the hardware.
<BR>(iv) Coordinating the many concurrent activities ensuring correct synchronisation
and communication is achieved.
<BR>(v) Acting as a Resource Guardian to protect various resources of the
computer system from malicious or accidental misuse.
<BR>(vi) Accounting for periods of resource usage by user processes, enforcing
quotas or restrictions as appropriate.
